# SelcukBolt Web

Bu klasor, mevcut Python Bolton mantigini kullanan Next.js + Tailwind tabanli web arayuzunu icerir.

## Gelistirme

```bash
cd web
npm install
npm run dev
```

API varsayilan olarak `http://127.0.0.1:8000` adresine baglanir. Gerekirse `.env.local` icinde `NEXT_PUBLIC_API_BASE_URL` tanimlayin.
